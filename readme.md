## UIU Healthcare
Eemergency doctor appointment for UIU Students. Student can search for available doctors. Doctors have their custom dashboard for submitting pescription online and get feedbacks, reports from student remotely. Student also have their custom dashboard where they can see which appointments they've taken which appointment to they've booked which doctors are available.

Technologies:
* React/Next.js
* PostgreSQL using Prisma with Neon hosting.

## Development

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or

```
